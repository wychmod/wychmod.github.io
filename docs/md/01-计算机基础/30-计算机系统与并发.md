### 1.3 计算机网络

> 📚 [计算机网络.md](../archive/old-cs-notes/计算机网络.md)

**OSI 7 层 / TCP-IP 4 层**：

```
应用层    HTTP / FTP / DNS / SSH
传输层    TCP / UDP
网络层    IP / ICMP / ARP
数据链路层  Ethernet / PPP
物理层     光纤 / 双绞线
```

**TCP 三次握手**：
```
Client → SYN → Server
Server → SYN + ACK → Client
Client → ACK → Server
```

**TCP 四次挥手**：
```
主动方 → FIN → 被动方
被动方 → ACK → 主动方
被动方 → FIN → 主动方
主动方 → ACK → 被动方
```

## 二、多线程与高并发

> 📚 [多线程与并发编程.md](../archive/old-cs-notes/多线程与并发编程.md)（38KB 完整版）

### 2.1 线程基础

- **线程 vs 进程**：线程共享进程内存，开销小
- **线程生命周期**：NEW / RUNNABLE / BLOCKED / WAITING / TIMED_WAITING / TERMINATED
- **线程调度**：抢占式 / 协作式
- **上下文切换**：保存 / 恢复寄存器

### 2.2 同步原语

| 工具 | 用途 | 性能 |
|---|---|---|
| **synchronized** | 互斥锁 | 中 |
| **ReentrantLock** | 可重入锁 | 高 |
| **ReadWriteLock** | 读写分离 | 高 |
| **StampedLock** | 乐观读 | 极高 |
| **Semaphore** | 信号量 | 中 |
| **CountDownLatch** | 倒计时门栓 | 中 |
| **CyclicBarrier** | 循环栅栏 | 中 |
| **Phaser** | 分阶段同步 | 中 |

### 2.3 线程池

```java
ThreadPoolExecutor executor = new ThreadPoolExecutor(
    2,  // corePoolSize
    10, // maximumPoolSize
    60, // keepAliveTime
    TimeUnit.SECONDS,
    new LinkedBlockingQueue<>(100),
    new ThreadPoolExecutor.CallerRunsPolicy()
);
```

### 2.4 并发集合

- **ConcurrentHashMap**：分段锁 + CAS
- **CopyOnWriteArrayList**：写时复制
- **ConcurrentLinkedQueue**：无锁队列
- **BlockingQueue**：阻塞队列

### 2.5 2026 高并发方案

- **Java 21 虚拟线程**（Loom）：百万级并发
- **Reactive**：Reactor / RxJava / Mutiny
- **协程**：Kotlin / Quarkus
- **Go 协程**：Golang 原生
- **Rust async**：零成本抽象
