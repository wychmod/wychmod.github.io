## 二、复杂度分析

### 2.1 时间复杂度

| 复杂度 | 名称 | 常见场景 |
|---|---|---|
| O(1) | 常数 | 数组随机访问、哈希表查询 |
| O(log n) | 对数 | 二分查找、堆操作、平衡树 |
| O(n) | 线性 | 遍历数组、链表 |
| O(n log n) | 线性对数 | 归并、快速、堆排序 |
| O(n²) | 平方 | 冒泡、选择、插入排序 |
| O(2ⁿ) / O(n!) | 指数/阶乘 | 暴力递归、全排列 |

### 2.2 空间复杂度

- **O(1)**：原地排序（冒泡、选择、插入、快排、堆排）。
- **O(n)**：归并排序临时数组、BFS 队列、DFS 递归栈。
- **O(n²)**：Floyd-Warshall 距离矩阵、部分 DP 表。

### 2.3 Master 公式

对于 T(N) = a·T(N/b) + O(Nᵈ)：

- log_b(a) < d → O(Nᵈ)
- log_b(a) = d → O(Nᵈ · log N)
- log_b(a) > d → O(N^{log_b(a)})

> 💡 补充：快排平均 O(n log n)，最坏 O(n²)；随机化快排通过随机选取 pivot 将期望复杂度稳定在 O(n log n)。

### 3.2 栈与队列

栈后进先出，用于括号匹配、表达式求值、DFS；队列先进先出，用于 BFS、滑动窗口。

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = [float('inf')]
    def push(self, x):
        self.stack.append(x)
        self.min_stack.append(min(x, self.min_stack[-1]))
    def pop(self):
        self.stack.pop(); self.min_stack.pop()
    def top(self): return self.stack[-1]
    def getMin(self): return self.min_stack[-1]
```

### 3.3 哈希表

哈希表提供近似 O(1) 的查询、插入、删除。Python 中 `dict` / `set` 最常用。典型应用：两数之和、复制带随机指针的链表、判断重复元素。

### 3.4 堆

堆是完全二叉树，数组实现。下标从 0 开始：左孩子 `2*i+1`，右孩子 `2*i+2`，父节点 `(i-1)//2`。

```python
def heapify(arr, index, size):
    left = index * 2 + 1
    while left < size:
        largest = left + 1 if left + 1 < size and arr[left+1] > arr[left] else left
        largest = largest if arr[largest] > arr[index] else index
        if largest == index: break
        arr[largest], arr[index] = arr[index], arr[largest]
        index, left = largest, largest * 2 + 1
```

> 💡 补充：从下往上建堆时间复杂度为 O(n)，从上往下为 O(n log n)；工程上小数据量用插入排序，大数据量用快排/堆排/归并。

## 四、排序算法

### 4.1 比较排序

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定 |
|---|---|---|---|---|
| 冒泡 | O(n²) | O(n²) | O(1) | 是 |
| 选择 | O(n²) | O(n²) | O(1) | 否 |
| 插入 | O(n²) | O(n²) | O(1) | 是 |
| 归并 | O(n log n) | O(n log n) | O(n) | 是 |
| 快速 | O(n log n) | O(n²) | O(log n) | 否 |
| 堆 | O(n log n) | O(n log n) | O(1) | 否 |

```python
# 归并排序
def merge_sort(arr, L, R):
    if L == R: return
    mid = L + (R - L) // 2
    merge_sort(arr, L, mid)
    merge_sort(arr, mid + 1, R)
    tmp, p1, p2 = [], L, mid + 1
    while p1 <= mid and p2 <= R:
        if arr[p1] <= arr[p2]: tmp.append(arr[p1]); p1 += 1
        else: tmp.append(arr[p2]); p2 += 1
    tmp.extend(arr[p1:mid+1]); tmp.extend(arr[p2:R+1])
    arr[L:R+1] = tmp
```

```python
# 快速排序（荷兰国旗优化）
import random

def quick_sort(arr, L, R):
    if L >= R: return
    arr[R], arr[random.randint(L, R)] = arr[random.randint(L, R)], arr[R]
    eq = partition(arr, L, R)
    quick_sort(arr, L, eq[0] - 1)
    quick_sort(arr, eq[1] + 1, R)

def partition(arr, L, R):
    if L == R: return [L, R]
    less, more, i = L - 1, R, L
    while i < more:
        if arr[i] < arr[R]: less += 1; arr[less], arr[i] = arr[i], arr[less]; i += 1
        elif arr[i] == arr[R]: i += 1
        else: more -= 1; arr[more], arr[i] = arr[i], arr[more]
    arr[more], arr[R] = arr[R], arr[more]
    return [less + 1, more]
```

### 4.2 非比较排序

**计数排序**：适用于整数且范围较小。统计每个值出现次数，再按顺序写回。

**基数排序**：按位分桶，常用于正整数。

## 五、查找与搜索

### 5.1 二分查找

二分查找要求数据有序，核心是利用二段性缩小搜索区间。

```python
def binary_search(nums, target):
    L, R = 0, len(nums) - 1
    while L <= R:
        mid = L + ((R - L) >> 1)
        if nums[mid] == target: return mid
        elif nums[mid] < target: L = mid + 1
        else: R = mid - 1
    return -1
```

**左右边界**：左边界在 `nums[mid] >= target` 时收紧右边界；右边界在 `nums[mid] <= target` 时收紧左边界。

### 5.2 DFS 与 BFS

DFS 用栈或递归，适合连通性、全排列、回溯；BFS 用队列，第一次到达目标即为最短路径（无权图）。

```python
from collections import deque

def bfs(start):
    q, visited = deque([start]), {start}
    while q:
        cur = q.popleft()
        for nxt in cur.nexts:
            if nxt not in visited:
                visited.add(nxt); q.append(nxt)
```

### 5.3 回溯

回溯通过“做选择 → 递归 → 撤销选择”枚举所有可能。

```python
# N 皇后
def solveNQueens(n):
    res, record = [], [-1] * n
    def valid(r, c):
        for i in range(r):
            if record[i] == c or abs(i - r) == abs(record[i] - c):
                return False
        return True
    def dfs(r):
        if r == n: res.append(record[:]); return
        for c in range(n):
            if valid(r, c):
                record[r] = c; dfs(r + 1)
    dfs(0); return res
```

> 💡 补充：位运算可将 N 皇后的常数时间大幅优化，用 `limit & (~(col | left | right))` 得到可用位置。

## 六、动态规划

### 6.1 DP 三要素

1. **重叠子问题**：重复计算可通过记忆化或 DP 表避免。
2. **最优子结构**：子问题最优能推出原问题最优。
3. **状态转移方程**：描述状态如何转移。

### 6.2 解题框架

```
初始化 base case
for 状态1 in 状态1所有取值:
    for 状态2 in 状态2所有取值:
        dp[状态1][状态2] = 最值(选择1, 选择2, ...)
```

### 6.3 常见 DP 模型

```python
# 0/1 背包
def knapsack(w, v, bag):
    n = len(w)
    dp = [[0] * (bag + 1) for _ in range(n + 1)]
    for i in range(n - 1, -1, -1):
        for rest in range(bag + 1):
            p1 = dp[i+1][rest]
            p2 = v[i] + dp[i+1][rest - w[i]] if rest >= w[i] else 0
            dp[i][rest] = max(p1, p2)
    return dp[0][bag]

# 最长公共子序列
def lcs(a, b):
    dp = [[0] * (len(b) + 1) for _ in range(len(a) + 1)]
    for i in range(1, len(a) + 1):
        for j in range(1, len(b) + 1):
            dp[i][j] = dp[i-1][j-1] + 1 if a[i-1] == b[j-1] else max(dp[i-1][j], dp[i][j-1])
    return dp[-1][-1]

# 凑零钱
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1); dp[0] = 0
    for coin in coins:
        for rest in range(coin, amount + 1):
            dp[rest] = min(dp[rest], dp[rest - coin] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1
```

> 💡 补充：暴力递归改 DP 的关键是“参数组合结构化”：先写暴力递归，找到可变参数，再画 DP 表按依赖顺序填表。空间压缩只保留必要行可进一步降低空间复杂度。

## 七、字符串与贪心

### 7.1 双指针与滑动窗口

滑动窗口用于子数组、子串问题，维护窗口内满足条件的元素。

```python
def minSubArrayLen(target, nums):
    L, total, res = 0, 0, float('inf')
    for R in range(len(nums)):
        total += nums[R]
        while total >= target:
            res = min(res, R - L + 1)
            total -= nums[L]; L += 1
    return res if res != float('inf') else 0
```

### 7.2 前缀和

前缀和 `pres[i]` 表示前 i 个元素之和，子数组 `[i, j)` 和为 `pres[j] - pres[i]`。结合同余定理可解决“子数组和能被 K 整除”等问题。

### 7.3 单调栈

单调栈用于寻找“下一个更大/更小元素”。

```python
def nextGreaterElements(T):
    stack, ans = [], [0] * len(T)
    for i in range(len(T)):
        while stack and T[i] > T[stack[-1]]:
            idx = stack.pop(); ans[idx] = i - idx
        stack.append(i)
    return ans
```

### 7.4 KMP

KMP 通过 `next` 数组避免主串指针回退。

```python
def strStr(s, p):
    if not p: return 0
    nxt = [0] * len(p)
    j = 0
    for i in range(1, len(p)):
        while j and p[i] != p[j]: j = nxt[j-1]
        if p[i] == p[j]: j += 1; nxt[i] = j
    j = 0
    for i in range(len(s)):
        while j and s[i] != p[j]: j = nxt[j-1]
        if s[i] == p[j]: j += 1
        if j == len(p): return i - j + 1
    return -1
```

### 7.5 贪心

贪心每一步做局部最优选择，难点在于证明全局最优。

```python
# 区间调度：按结束时间排序
def best_arrange(programs):
    programs.sort(key=lambda x: x.end)
    timeline = res = 0
    for p in programs:
        if timeline <= p.start:
            res += 1; timeline = p.end
    return res

# 分金子（哈夫曼）
import heapq
def less_money(arr):
    heapq.heapify(arr); total = 0
    while len(arr) > 1:
        cur = heapq.heappop(arr) + heapq.heappop(arr)
        total += cur; heapq.heappush(arr, cur)
    return total
```

## 八、树与图

### 8.1 二叉树遍历

```python
# 中序非递归
def inorder(root):
    stack, res = [], []
    while root or stack:
        while root: stack.append(root); root = root.left
        root = stack.pop(); res.append(root.val); root = root.right
    return res

# 层序
def levelOrder(root):
    from collections import deque
    if not root: return []
    q, res = deque([root]), []
    while q:
        level = []
        for _ in range(len(q)):
            node = q.popleft(); level.append(node.val)
            if node.left: q.append(node.left)
            if node.right: q.append(node.right)
        res.append(level)
    return res
```

### 8.2 BST 与 LCA

BST 中序遍历有序。普通二叉树 LCA：

```python
def lowestCommonAncestor(root, p, q):
    if not root or root == p or root == q: return root
    left = lowestCommonAncestor(root.left, p, q)
    right = lowestCommonAncestor(root.right, p, q)
    return root if left and right else (left or right)
```

### 8.3 Trie

```python
class Trie:
    def __init__(self):
        self.children = {}
        self.is_end = False
    def insert(self, word):
        node = self
        for ch in word:
            node = node.children.setdefault(ch, Trie())
        node.is_end = True
    def search(self, word):
        node = self
        for ch in word:
            if ch not in node.children: return False
            node = node.children[ch]
        return node.is_end
```

### 8.4 并查集

```python
class UF:
    def __init__(self, n):
        self.parent = list(range(n))
        self.size = [1] * n
        self.cnt = n
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    def union(self, p, q):
        a, b = self.find(p), self.find(q)
        if a == b: return
        if self.size[a] < self.size[b]: a, b = b, a
        self.parent[b] = a; self.size[a] += self.size[b]; self.cnt -= 1
    def connected(self, p, q): return self.find(p) == self.find(q)
```

### 8.5 图算法

**拓扑排序**：不断移除入度为 0 的节点。

```python
# Dijkstra（堆优化）
import heapq

def dijkstra(graph, start):
    dist = {start: 0}; heap = [(0, start)]
    while heap:
        cost, u = heapq.heappop(heap)
        if cost > dist.get(u, float('inf')): continue
        for v, w in graph.get(u, []):
            if cost + w < dist.get(v, float('inf')):
                dist[v] = cost + w; heapq.heappush(heap, (dist[v], v))
    return dist

# Floyd-Warshall
def floyd_warshall(g, n):
    dist = [row[:] for row in g]
    for k in range(n):
        for i in range(n):
            for j in range(n):
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
    return dist
```

> 💡 补充：图算法代码量较大，面试时建议先用最熟悉的结构（如邻接表）表达图，再套用模板；Dijkstra 不能处理负权边，负权请用 Bellman-Ford 或 SPFA。

## 十、蓝桥杯与竞赛

### 10.1 数据范围反推算法

蓝桥杯常通过数据规模判断可用算法：

- n ≤ 20：暴力、状压 DP。
- n ≤ 100：O(n³) DP。
- n ≤ 1000：O(n²) 动态规划、Floyd。
- n ≤ 10⁵：O(n log n) 二分、排序、线段树、树状数组。
- n ≤ 10⁶：O(n) 线性算法。

## 十二、常见坑与补充

### 12.1 排序

1. 归并排序可用“内部缓存法”做到 O(1) 空间，但会失去稳定性。
2. “原地归并排序”某些实现可能退化到 O(n²)。
3. 快排稳定性改进（01 stable sort）对数据分布有额外要求。
4. 在 O(n) 时间、O(1) 空间内保持奇偶相对顺序，一般不可行。

### 12.2 二分查找

- 区间开闭决定 `while` 条件和边界更新方式。
- 求 `mid` 时若用 `L = mid`，需改为 `mid = L + ((R - L) >> 1) + 1` 防止死循环。
- Python 可用 `bisect_left` / `bisect_right` 简化边界查找。

### 12.3 动态规划

- 先写暴力递归，再改记忆化搜索，最后改迭代 DP。
- 可变参数尽量简单，最好不超过 int；若为一维线性结构，确保是唯一可变参数。
- 注意初始化与遍历顺序，避免越界。

### 12.4 链表与树

- 链表操作务必处理空节点和尾节点指向。
- 树的后序遍历常用来“先收集子树信息，再整合父节点信息”。
- 序列化时要显式保存空节点，避免歧义。

### 12.5 图与并查集

- 有向图和无向图都可以用有向图表达。
- 并查集路径压缩 + 按秩合并可将均摊复杂度降到近似 O(α(n))。
- Dijkstra 不能处理负权边。

> 💡 补充：lb算法小抄笔记强调“数据结构是工具，算法是方法”，建议从二叉树入手培养递归框架思维，因为大部分算法技巧本质都是树的遍历问题。左神笔记则把“暴力递归 → 记忆化搜索 → 经典 DP”作为掌握动态规划的最佳路径。
