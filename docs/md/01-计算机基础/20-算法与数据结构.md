## 一、算法笔记精选

### 1.1 动态规划（核心）

- 状态定义
- 状态转移方程
- 初始化
- 遍历顺序
- 返回值

**经典题目**：
- 斐波那契 / 爬楼梯
- 背包问题（01 背包 / 完全背包）
- 最长公共子序列（LCS）
- 最长上升子序列（LIS）
- 编辑距离
- 股票买卖

### 1.2 二叉树

- 遍历（前序 / 中序 / 后序 / 层序）
- BFS / DFS
- 递归 / 迭代
- 树的序列化 / 反序列化

### 1.3 链表

- 反转链表
- 合并有序链表
- 链表环检测
- 链表相交

### 1.4 排序

| 算法 | 时间 | 空间 | 稳定 |
|---|---|---|---|
| 冒泡 | O(n²) | O(1) | ✓ |
| 选择 | O(n²) | O(1) | ✗ |
| 插入 | O(n²) | O(1) | ✓ |
| 归并 | O(n log n) | O(n) | ✓ |
| 快排 | O(n log n) | O(log n) | ✗ |
| 堆排 | O(n log n) | O(1) | ✗ |
| 计数 | O(n+k) | O(k) | ✓ |
| 基数 | O(n*k) | O(n+k) | ✓ |

### 1.5 搜索

- 二分查找（前提：有序）
- BFS（最短路）
- DFS（连通性）
- A*（启发式）
