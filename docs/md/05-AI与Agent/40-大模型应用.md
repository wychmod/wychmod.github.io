| 时间 | 里程碑 |

### 1.2 关键技术

- **Transformer**：注意力机制是核心
- **预训练（Pre-training）**：大规模无监督学习
- **微调（Fine-tuning）**：LoRA / QLoRA / PEFT
- **RLHF / DPO**：人类反馈对齐
- **RAG**：检索增强生成
- **Function Calling / Tool Use**：让 LLM 调用工具
- **长上下文**：RoPE / YaRN / 滑动窗口注意力

## 二、大模型开发基础：Embedding

> 📚 [2-大模型开发基础：OpenAI Embedding.md](../../archive/old-llm-notes/gpt大模型应用/2-大模型开发基础：OpenAI%20Embedding.md)（3KB）

**Embedding 是什么**：
- 把文本映射到高维向量空间
- 语义相近的文本 → 向量距离近
- 用于：检索 / 聚类 / 推荐 / 分类

**主流 Embedding 模型**：
- OpenAI `text-embedding-3-small` / `text-embedding-3-large`
- BGE（BAAI 通用中文）
- M3E（Mixed Model Efficient Embedding）
- Cohere embed-multilingual-v3
- Qwen Embedding

```python
from openai import OpenAI
client = OpenAI()

response = client.embeddings.create(
    model="text-embedding-3-small",
    input="你好世界"
)
vec = response.data[0].embedding  # 1536 维向量
```

## 四、2026 年 NLP 主流方案

> ⚠️ 传统 NLP（贝叶斯/HMM/word2vec）已被 LLM 全面替代

```
方案优先级：
1. LLM API（GPT-4 / Claude / Qwen / DeepSeek）
2. 预训练 Embedding（BGE / M3E / Qwen Embedding）
3. RAG（LangChain / LlamaIndex）
4. Agent（LangGraph / AutoGen）

```
