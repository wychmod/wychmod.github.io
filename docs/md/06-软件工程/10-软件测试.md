## 一、测试基础

### 1.1 软件测试的核心目标

> 📚 完整笔记见 [软件测试基础知识.md](../../archive/old-testing-notes/软件测试基础知识.md)（已归档）

- **验证功能正确性**：需求是否被满足
- **发现缺陷**：尽早暴露问题
- **评估质量**：覆盖率、性能、稳定性
- **提供决策依据**：是否能发布

### 1.2 测试分类

| 维度 | 分类 |
|---|---|
| **阶段** | 单元测试 / 集成测试 / 系统测试 / 验收测试 |
| **方式** | 手动测试 / 自动化测试 |
| **视角** | 黑盒 / 白盒 / 灰盒 |
| **目标** | 功能测试 / 性能测试 / 安全测试 / 兼容性测试 |
| **执行** | 静态测试 / 动态测试 |

### 1.3 测试用例设计方法

> 📚 完整笔记见 [测试用例设计方法.md](../../archive/old-testing-notes/测试用例设计方法.md)（已归档）

| 方法 | 说明 |
|---|---|
| **等价类划分** | 输入域划分为有效/无效等价类，从每类选代表 |
| **边界值** | 取边界值（最小/略小/正常/略大/最大） |
| **因果图** | 输入条件与结果的逻辑关系 |
| **判定表** | 复杂条件组合 |
| **正交实验** | 多因素多水平的代表组合 |
| **场景法** | 用户场景驱动 |
| **错误猜测** | 经验判断容易出错的地方 |

### 1.4 执行用例

> 📚 完整笔记见 [执行测试用例.md](../../archive/old-testing-notes/执行测试用例.md)（已归档）

## 三、代码覆盖率工具 JaCoCo

> 📚 完整笔记见 [jacoco工具.md](../../archive/old-testing-notes/jacoco工具.md)（已归档）

### 3.1 集成方式

```xml
<!-- pom.xml -->
<plugin>
    <groupId>org.jacoco</groupId>
    <artifactId>jacoco-maven-plugin</artifactId>
    <version>0.8.11</version>
    <executions>
        <execution>
            <goals>
                <goal>prepare-agent</goal>
            </goals>
        </execution>
        <execution>
            <id>report</id>
            <phase>test</phase>
            <goals>
                <goal>report</goal>
            </goals>
        </execution>
    </executions>
</plugin>
```

### 3.2 查看报告

```bash
mvn clean test
# 报告路径: target/site/jacoco/index.html
```

### 3.3 覆盖率指标

| 指标 | 含义 |
|---|---|
| **行覆盖** | 至少执行一次的代码行 |
| **分支覆盖** | if/else 都被执行 |
| **方法覆盖** | 至少被调用一次的方法 |
| **类覆盖** | 至少被加载的类 |

## 四、Chrome 插件设计

> 📚 完整笔记见 [测试助手chrome插件设计文档.md](../../archive/old-testing-notes/测试助手chrome插件设计文档.md)（已归档）

### 4.1 Manifest V3 基础

```json
{
  "manifest_version": 3,
  "name": "Test Helper",
  "version": "1.0",
  "permissions": ["activeTab", "storage"],
  "host_permissions": ["<all_urls>"],
  "background": {
    "service_worker": "background.js"
  },
  "content_scripts": [{
    "matches": ["<all_urls>"],
    "js": ["content.js"]
  }]
}
```

### 4.2 Chrome API 常用

- `chrome.tabs` — 标签页操作
- `chrome.storage` — 本地存储
- `chrome.runtime` — 消息传递
- `chrome.contextMenus` — 右键菜单
