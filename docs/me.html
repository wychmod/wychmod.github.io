🤖 openai-gateway · ChatGPT 网关服务

AI · Featured 实习

背景:ChatGPT 开放初期对国区 IP 和爬虫倾向的 API 调用进行封禁,为满足算法团队对大模型调研的迫切需求, 从零搭建一个对接飞书机器人、带 OAuth 身份验证、多 API Key 配置筛选、OpenAI 转发、问答内容存储的中转网关。

Java Spring Boot OpenAI API Redis RabbitMQ Feishu OAuth Docker
难点 · 解决方案 国区 IP 拦截 + 多种反爬风控导致单 key 不可用。 解决:多 key 池 + 滑动窗口健康度打分 + 失败 key 临时熔断, 配合会话级 hash 路由,既保证稳定性又方便后续按会话做去重与统计。
github.com/wychmod/openai-gateway README ⭐ 团队建议开源
