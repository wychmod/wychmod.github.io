Harness · 核心竞争力 方法论 13 章节 CLAUDE.md

以"复刻 Dify 的极简版 AI Agent 平台 Hify"为载体,沉淀一套可被任意团队复用的 AI 辅助开发 Harness / 上下文工程模板。 完整跑通"业界标杆研究 → 功能取舍 → 技术选型 → 架构 → 代码组织 → 部署 → 性能 → 扩展"13 个章节, 让 AI 拿到 Harness 即可直接产出可上线代码。
