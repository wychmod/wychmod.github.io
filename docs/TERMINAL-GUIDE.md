## 💡 使用技巧

### 1. 快速导航
```bash
# 查看所有分类
$ ls

# 进入某个分类
$ cd 后端开发

# 查看当前分类下的内容
$ ls

# 查看子分类
$ cd Java技术栈
$ ls

# 返回上一级
$ cd ..
```

### 2. 文档搜索
```bash
# 搜索包含 "Redis" 的所有文档
$ find Redis

# 搜索包含 "Spring" 的文档
$ grep Spring

# 在特定目录下查找
$ cd 后端开发
$ find MySQL
```

### 3. 查看文档
```bash
# 直接打开文档（会跳转到文档页面）
$ cat Java基本功

# 也可以使用别名
$ open Spring源码解析
$ view Python高级编程
```

### 4. 目录树查看
```bash
# 查看整个知识库结构
$ tree

# 查看特定分类的结构
$ cd 人工智能
$ tree
```

### 5. 获取帮助
```bash
# 查看所有命令
$ help

# 查看特定命令的详细说明
$ man ls
$ man cd
$ man find
```

### 6. 命令历史
```bash
# 查看历史命令
$ history

# 使用 ↑ 和 ↓ 快速调用历史命令
# 按 ↑ 键会显示上一条命令
# 按 ↓ 键会显示下一条命令
```

## 🌟 新增功能亮点

### 1. 增强的 ls 命令
- ✨ 显示图标：目录用 📁，文件用 📄
- 📊 统计信息：显示目录和文件的数量
- 🎨 颜色区分：目录显示金色，文件显示绿色

示例输出：
```
$ ls 后端开发
📁 Java技术栈/
📁 Python技术栈/
📁 Go语言/

总计: 3 个目录, 0 个文件
```

### 2. man 命令手册系统
- 📖 提供命令的详细说明
- 📝 包含语法、描述、示例
- 💡 帮助快速掌握命令用法

示例：
```
$ man ls
NAME
    ls - list directory contents

SYNOPSIS
    ls [path]

DESCRIPTION
    List information about files and directories.
    By default, lists current directory.

EXAMPLES
    ls              List current directory
    ls 后端开发      List specific directory
```

### 3. 多个命令别名
- `ll` = `ls`（Linux 风格）
- `dir` = `ls`（Windows 风格）
- `open` = `cat`（macOS 风格）
- `view` = `cat`（Vim 风格）
- `quit` = `exit`

### 4. grep 搜索命令
- 🔍 快速搜索功能
- 🎯 模糊匹配文档标题
- 📋 显示完整路径
