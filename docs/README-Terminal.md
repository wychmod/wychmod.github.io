# 命令行终端使用指南

## 功能简介

本技术知识库现已升级为**黑金色调**主题，并集成了一个强大的**命令行终端导航系统**，为您提供两种并行的文档浏览方式：

- 🖱️ **传统导航**：侧边栏、顶部导航栏
- ⌨️ **命令行导航**：类 Unix 终端风格的命令行界面

## 打开终端

### 方式一：点击悬浮按钮
点击页面右下角的金色圆形按钮（►_）

### 方式二：键盘快捷键
- Windows/Linux: `Ctrl + K`
- macOS: `Cmd + K`

## 基础命令

### 文档导航命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `ls` | 列出当前目录内容 | `ls` |
| `ls <路径>` | 列出指定目录内容 | `ls 后端开发` |
| `pwd` | 显示当前路径 | `pwd` |
| `cd <目录>` | 切换到指定目录 | `cd Java技术栈` |
| `cd ..` | 返回上级目录 | `cd ..` |
| `cd /` | 返回根目录 | `cd /` |
| `cat <文件>` | 打开指定文档 | `cat Spring源码解析` |
| `find <关键词>` | 搜索包含关键词的文档 | `find Redis` |
| `tree` | 显示目录树状结构 | `tree` |

### 系统命令

| 命令 | 说明 |
|------|------|
| `help` | 显示帮助信息 |
| `history` | 查看命令历史 |
| `clear` | 清空终端屏幕 |
| `whoami` | 显示用户信息 |
| `about` | 显示知识库统计信息 |

### 彩蛋命令

| 命令 | 说明 |
|------|------|
| `fortune` | 获取随机技术名言 |
| `neofetch` | 显示系统信息面板 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|

| `Esc` | 关闭终端 |

| `↑` / `↓` | 浏览历史命令 |

## 使用示例

### 示例 1：浏览 Java 技术栈文档

```bash
$ cd 后端开发
$ cd Java技术栈
$ ls
$ cat Java基本功
```

### 示例 2：搜索 Redis 相关文档

```bash
$ find Redis
$ cd 数据存储与中间件/Redis缓存
$ ls
```

### 示例 3：查看文档树结构

```bash
$ cd /
$ tree
$ cd 人工智能
$ tree
```

## 路径规则

- **绝对路径**：以 `/` 开头，如 `/后端开发/Java技术栈`
- **相对路径**：从当前目录开始，如 `Java技术栈`
- **上级目录**：使用 `..` 返回上一级
- **根目录**：使用 `/` 或直接 `cd` 无参数

## 特色功能

### 1. 智能搜索
使用 `find` 命令可以模糊搜索文档标题，快速定位您需要的内容。

### 2. 命令历史
系统会自动保存您的命令历史（最多 100 条），下次访问时仍然可用。

### 3. 自动补全
输入命令的部分内容后按 `Tab` 键，系统会尝试自动补全。

### 4. 树状视图
使用 `tree` 命令可以清晰地看到文档的层级结构。

## 技术实现

- **框架**：基于 Docsify 4.x
- **样式**：纯 CSS，无外部依赖
- **脚本**：原生 JavaScript，轻量高效
- **存储**：使用 localStorage 持久化命令历史
- **兼容性**：支持现代浏览器（Chrome、Firefox、Safari、Edge）

## 注意事项

1. 命令和路径支持中文
2. 路径名称不区分大小写
3. 部分命令支持模糊匹配
4. 移动端访问时终端会自动适配屏幕尺寸
